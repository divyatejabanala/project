To-Do List Application

This project is a user-friendly To-Do List Application built using HTML, CSS, and JavaScript. It allows users to add, edit, delete, and mark tasks as completed. The interface is responsive, interactive, and designed with dynamic DOM manipulation.

Frontend HTML, CSS, JavaScript
Key Features
  • Add, edit, and delete tasks in real-time
  • Mark tasks as completed with visual indicators
  • Tasks persist across sessions using Local Storage
  • Fully responsive layout with smooth user experience

This application demonstrates practical knowledge in DOM manipulation, local storage, and UI/UX design principles. It's ideal for showcasing JavaScript capabilities in personal productivity tools.
